# DripDrop

A simple web page that shows falling rain with sound effects.

## Demo

[![Watch Demo](https://img.youtube.com/vi/OOVCSwfQvCk/0.jpg)](https://www.youtube.com/watch?v=OOVCSwfQvCk)

## Features

- Animated rain drops with depth effect  
- Two modes: Light Rain and Stormy Rain...planning to add more, idk when :)   
- Background rain sound with volume control  
- Click anywhere to play/pause audio  
- Minimal UI with auto-hide behavior

- HTML
- CSS
- JavaScript
